Detailed Description

Represents a global configuration (rd_kafka_conf_t).

This wraps an rdkafka configuration handle. It can safely be copied (will use rd_kafka_conf_dup under the hood) and moved.

Some other overloads for Configuration::set are given via ConfigurationBase.
