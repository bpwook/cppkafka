Detailed Description

Represents a view of a buffer.

This is only a view, hence you should convert the contents of a buffer into some other container if you want to store it somewhere.

If you're using this to produce a message, you need to guarantee that the pointer that this buffer points to will still until the call to Producer::produce returns.
