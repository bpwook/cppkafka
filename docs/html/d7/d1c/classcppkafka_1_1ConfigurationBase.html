Gets a value, converting it to the given type.

If the configuration option is not found, then ConfigOptionNotFound is thrown.

If the configuration value can't be converted to the given type, then InvalidConfigOptionType is thrown.

Valid conversion types:

  • std::string
  • bool
  • int
