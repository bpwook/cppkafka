Detailed Description

Represents a rdkafka queue.

This is a simple wrapper over a rd_kafka_queue_t*

Queue cppkafka::Queue::make_queue ( rd_kafka_queue_t *  handle)
static

\brieef Creates a Queue object out of a handle.

This will check what the rdkafka version is and will return either an owned queue handle or a non owned one, depending on whether the current version is >= RD_KAFKA_QUEUE_REFCOUNT_BUG_VERSION (see macros.h)

handleThe handle to be used
